# Beer Delivery

A static browser game. Open `index.html` directly, or deploy the folder as a static site.

## Files needed online

- `index.html`
- `WebAssets/beer_sprites_transparent.png`

## Fast deploy options

### Netlify Drop

1. Open https://app.netlify.com/drop
2. Drag this project folder into the page.
3. Netlify will give you a public URL.

### Vercel

Deploy this folder as a static project. No build command is needed.

### GitHub Pages

Push this folder to a GitHub repository, then enable Pages from the repository settings.
